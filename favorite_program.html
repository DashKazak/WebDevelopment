The time I wrote an inventory application in C#

My C# Invenory Maintnence program is able to keep track of your inventory. The inputs are customized to fit your needs for plants/ garden supplies (may vary), so no more unnecessary typing! You are in complete control: you can add and delete the list to stay up-to-date. Happy inventoring!!

Here is a piece of code that validates your input and decided what information to show next: 

private void btnSave_Click(object sender, EventArgs e)
            {
                if (IsValidData())
                {
                    if (rdioPlant.Checked)
                    {
                        item = new Plant(txtItemNo.Text, txtSize.Text, txtDescription.Text, txtPrice.Text);
                    }
                    if (rdioSupply.Checked)
                    {
                        item = new Supply(txtItemNo.Text, cmbManifact.Text, txtDescription.Text, txtPrice.Text);
                    }
                    
                    this.Close();
                }
            }
